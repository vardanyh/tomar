# Tomar (Tomarakan Method)

This project is a modernization and refactoring of the original **Tomar** application, a DOS-based system built using **Clipper**. It implements the **Tomarakan eghanak** (Armenian calendar method), a unique system invented by **Rafik Vardanyan**.
